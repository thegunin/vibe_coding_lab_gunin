Уточни промпт ниже, чтобы бот был сделан точно хорошо

Создай Telegram-бота на Python с использованием библиотеки python-telegram-bot.

Функционал бота:

ОБЩЕЕ ОПИСАНИЕ

Бот для сбора обратной связи

Собирает мнения команды по вопросам
Формирует сводные отчеты
Отправляет уведомления руководителю
Ведет статистику

УТОЧНЕНИЕ 

Для понятной визуализации функционала давай создадим четыре аккаунта: 
один аккаунт руководителя (логин manager, пароль manager-password) 
и три аккаунта сотрудников (логин employee-1, пароль employee-1) и так далее для всех трех 

С аккаунта руководителя можно создавать вопросы для команды (каждый вопрос предполагает текст с открытой формулировкой вопроса и кнопки от 1 до 5 с вариантами ответа) 

У сотрудников во вкладке вопросы появляются на выбор неотвеченные вопросы, при выборе вопроса – отображается полный текст и возможность кликнуть на ответ от 1 до 5

При клике руководителю отправляется уведомление (давай сделаем отдельную кнопку история уведомлений) 

Плюс давай сделаем кнопку "Отчеты" со средним баллом и количеством ответивших по каждому вопросу 



Требования:
- Бот должен быть простым и понятным
- Код должен быть хорошо прокомментирован
- Использовать файл для хранения данных (JSON или SQLite)
- Добавить обработку ошибок

Создай:

1. Файл bot.py с кодом бота
2. Файл requirements.txt с зависимостями
3. Файл README.md с инструкцией по запуску
4. Файл .env.example для примера конфигурации
